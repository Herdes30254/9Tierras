----COMO ABRIR EL PROYECTO LOCALMETE-----
Requisitos previos

Asegúrate de tener instalado:

Node.js (versión 14 o superior)
https://nodejs.org

npm (se instala junto con Node)

MongoDB
Puedes usar:

MongoDB local (instalado en tu PC), o MongoDB Atlas (base de datos en la nube).

Luego abrir el CMD como administrador y escribir "cd ruta/del/proyecto"
Luego "npm install"
Luego "npm start"

Una vez iniciado el servidor, abre en tu navegador:
http://localhost:3000
